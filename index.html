About

Hi there! I'm Kim. I'm a software engineer who likes using technology to solve interesting problems. When I'm not at my computer, you can probably find me traveling, drinking copious amounts of coffee, or mastering the art of the pull-up.

I graduated with a B.A. in Computer Science from Dartmouth College in 2015, where I led the Women in Computer Science Club. Since graduating, I've been working in a rotational Program at Apple where I've worn many hats. I've tried out various areas of the stack as well as user experience.

I'm passionate about the intersection between technology and education and I love teaching others things I've learned along the way.

Projects

While I've worked on a number of cool projects at Apple in the past two years, they're secret! Here are some of the projects I've done in my free time.

30-20-10

After reading the New York Times article A Way to Get Fit and Have Fun, I built an iOS app in Swift to implement a fun, 30-20-10 style workout that would run in the background. This app lets you configure the settings of your workout and provides audio cues when it's time to switch to jogging, running, sprinting, resting, cooling down, etc. Because the app works in the background, you can listen to whatever music you'd like during the workout. Check out the Github repo.

Pixel Art

I recently built a pixel art generator using React and Redux, with undo and redo functionality. It was a lot of fun to see the power of Redux and functional programming. I plan to add a bucket-fill feature, image export capabilities, and greater user customization. Check out the Github repo.

GratBot

A number of studies have shown that incorporating a gratitude practice into your daily routine can make you happier. After reading a number of articles on the subject, I decided to make a chatbot that would make it easy to do this! I wrote this project using NodeJS, MongoDB, and the Messenger Webhook. Then I deployed it to Heroku and set up a Heroku Scheduler to automatically message users at their deteremined notification frequency. Check out the Github repo

Charlotte's Website

It's always been a goal of mine to write a children's book about Computer Science, and recently I figured, why wait?! After reading Julia Evans' Networking Zine, I've been inspired to create my own zine geared toward children about basic web development. Check out my work in progress.
